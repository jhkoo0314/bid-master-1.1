```md
## 1. 개요 및 개선 목적

BidMaster는 "실전 경매 훈련용 시뮬레이터"이지만, 현재 리포트 구조는 **정보는 맞지만 현실감과 실전성과 설명력이 부족**한 상태임.  
또한 "초보자도 이해할 수 있는 리스크 정보"가 부족하고, 일부 계산 지표(수익률, 점유, FMV 등)가 **UI와 용어 정의 측면에서 혼동을 유발**하고 있음.

본 문서는 다음 5가지 요소의 통합 개선을 목표로 한다:

1. 리포트 데이터 간 구성이 통일되고, 정보 단절이 없는 단일 분석 체계 구축
2. 사용자 입찰액과 수익률 해석에 오해가 없도록 **표시 기준·단위·라벨 명확화**
3. 경쟁자(입찰자) 생성 로직을 현실형으로 개선하여 "전체가 과열되는 오류" 제거
4. **점유 리스크를 AI로 자동 추정 + 수치화 + 명도비용까지 예측**하는 실전형 로직 도입
5. 리포트가 단순 정보 나열이 아닌 "투자 판단 훈련 도구"가 되도록 구조화

## 2. 리포트 공통 개선 사항

| 항목 | 기존 문제 | 개선 내용 |
|-------|-----------|-----------|
| FMV(예상 시세) 표시 | 경매분석 리포트에만 존재 | ✅ 3종 리포트 모두 동일 FMV 컴포넌트 적용 |
| 수익률 계산 라벨 | `+6.3%`처럼 분모 미표시 → 혼동 | ✅ "총투자금 대비 ROI" / "낙찰가 기준 수익률" 분리 표기 |
| 입찰 위험 경고 | 과열/저가 여부 기준 부족 | ✅ FMV 대비 과열 경고 + 경쟁자 수 증가 표시 |
| 권리 용어 난도 | "말소기준권리" 등 초보자 이해 어려움 | ✅ 용어 툴팁 + 권리 소멸/인수 표로 시각화 |
| 리포트 어조 | 교육형/사실형 혼재 | ✅ “사실 기반 + 경고 기반 + 교육적 해석” 3단 구성 |

## 3. 권리분석 리포트 개선

### ✅ 구조 변경 전/후

| 기존 | 개선 |
|------|------|
| 말소기준권리 1줄 요약 | 말소/인수 여부 테이블 + 해석 문장 포함 |
| "임차인 없음" 단정 표시 | AI 점유 예측 + 점유 리스크 점수 + 명도비용 추정 포함 |
| 세부 권리 목록만 나열 | "등기 순위 → 소멸 여부 → 인수 여부" 순으로 시각화 |
| FMV 표시 없음 | 동일 FMV 범위 자동 삽입 |

### ✅ 예시 개선 UI

```
[권리 요약]
말소기준권리: 근저당권 (2018-10-12)
말소권리: 4건
인수권리: 0건 ✅ (인수 위험 없음)

[권리 소멸/인수 분석]
| 권리 | 순위 | 소멸 | 인수 |
|------|------|------|------|
| 근저당 | 1순위 | ✅ | X |
| 가압류 | 2순위 | ✅ | X |
| 가처분 | 3순위 | ✅ | X |

[점유 및 명도 리스크]
AI 예측 점유 위험도: 68% (중간)
예상 명도 비용: 350만~500만원
⚠️ 실제 점유 상태는 매각물건명세서/현장 방문으로 확인 필요
```

## 4. 경매분석 리포트 개선

### ✅ 수익률 표기 방식 개선

| 기존 표기 | 문제점 |
|------------|---------|
| `+6.3%` | “낙찰가 기준 수익률”인지 “총투자금 대비 ROI”인지 구분 안 됨 |

✅ 개선 후:

```
수익률 분석

+6.3% (낙찰가 기준 수익률)
+17.1% (총투자금 대비 ROI)
-12.6% (FMV 기준 손실률)
```

```

```
## 6. 경쟁자 생성 로직 고도화

### ✅ 문제점 (현 상태)
- 사용자 입찰가가 높으면 경쟁자들도 따라 올라감 → 현실 반영 실패
- 경쟁자 수 고정값 → 과열 상황 반영 X

### ✅ 개선 후 로직 요약

```
기준가: FMV 중심 생성 (userBid 영향 없음)
과열도 = bidPrice / FMV
competitorCount += baseCount * (0.4 × 과열도)
입찰가 상한 = min(FMV*1.05, 감정가*0.99, userBid*0.985)
꼬리 구간: 상위 10%만 과열 반영
```

## 7. 임차인·점유 리스크 AI 고도화

### ✅ 구성 요소

| 요소 | 설명 |
|--------|-------|
| 점유 리스크 점수 | 0~100점, 3단계 라벨(낮음/중간/높음) |
| 명도비용 추정 | 200만~800만원 구간 예측 |
| 배당요구 여부 | 등록 시 인수 가능성 리스크 즉시 반영 |
| 판례 기반 가중치 | “보증금 > 배당금” 시 위험 +20% |

### ✅ 점유 리스크 산출 공식 (v1)

```
점유RiskScore =
(확정일자여부 × 0.25) +
(전입일시 선순위 여부 × 0.25) +
(배당요구 여부 × 0.20) +
(최근 판례 리스크 계수 × 0.15) +
(유찰 횟수 기반 점유 지속 가능성 × 0.15)
```

### ✅ 출력 UI 예시

```
[점유 리스크 분석]

점유 위험도: 72% (높음)
예상 명도비용: 380만~520만원
배당요구: 없음 (보증금 인수 가능성 있음)
* 실제 점유 상태 확인 필요 (명도 소송 가능성 있음)
